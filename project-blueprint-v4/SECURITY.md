# Security and Privacy

Documents may contain sensitive data.

Rules:
- Do not log raw OCR text unnecessarily.
- Restrict Admin access.
- Protect uploaded files.
- Keep audit logs for admin actions.
- Redact sensitive logs where possible.
- Allow document/OCR result deletion where feasible.

Admin actions to log:
- template created
- template page created
- field edited
- ignore region edited
- embedding generated
- template tested
- template approved
- template rejected
- template disabled
